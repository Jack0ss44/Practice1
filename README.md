–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—ã–±–æ—Ä–æ–º (Selection Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ: 
–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—ã–±–æ—Ä–æ–º ‚Äî —ç—Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º, –∫–æ—Ç–æ—Ä—ã–π –Ω–∞—Ö–æ–¥–∏—Ç –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –Ω–µ–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —á–∞—Å—Ç–∏ –º–∞—Å—Å–∏–≤–∞ –∏ –º–µ–Ω—è–µ—Ç –µ–≥–æ –º–µ—Å—Ç–∞–º–∏ —Å –ø–µ—Ä–≤—ã–º —ç–ª–µ–º–µ–Ω—Ç–æ–º —ç—Ç–æ–π —á–∞—Å—Ç–∏.

–û–ø–∏—Å–∞–Ω–∏–µ: 
–ù–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –∏—â–µ—Ç—Å—è –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å—Ä–µ–¥–∏ –Ω–µ–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∏ —Å—Ç–∞–≤–∏—Ç—Å—è –≤ –Ω–∞—á–∞–ª–æ –Ω–µ–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –ù–∞—á–∏–Ω–∞–µ–º —Å –ø–µ—Ä–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –º–∞—Å—Å–∏–≤–∞.
2. –ù–∞—Ö–æ–¥–∏–º –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ –Ω–µ–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —á–∞—Å—Ç–∏.
3. –ú–µ–Ω—è–µ–º –Ω–∞–π–¥–µ–Ω–Ω—ã–π –º–∏–Ω–∏–º—É–º —Å —Ç–µ–∫—É—â–∏–º —ç–ª–µ–º–µ–Ω—Ç–æ–º.
4. –£–≤–µ–ª–∏—á–∏–≤–∞–µ–º –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—É—é —á–∞—Å—Ç—å –Ω–∞ –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç.
5. –ü–æ–≤—Ç–æ—Ä—è–µ–º, –ø–æ–∫–∞ –≤–µ—Å—å –º–∞—Å—Å–∏–≤ –Ω–µ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω.

–°–ª–æ–∂–Ω–æ—Å—Ç—å:

–ü—Ä–∏–º–µ—Ä:
```python
def selection_sort(arr):
    for i in range(len(arr)):
        min_idx = i
        for j in range(i + 1, len(arr)):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

print(selection_sort([23, 56, 21, 41, 12]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[12, 21, 23, 41, 56]`



–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—É–∑—ã—Ä—å–∫–æ–º (Bubble Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ: 
–ê–ª–≥–æ—Ä–∏—Ç–º, –∫–æ—Ç–æ—Ä—ã–π –º–Ω–æ–≥–æ–∫—Ä–∞—Ç–Ω–æ –ø—Ä–æ—Ö–æ–¥–∏—Ç –ø–æ –º–∞—Å—Å–∏–≤—É –∏ –º–µ–Ω—è–µ—Ç –º–µ—Å—Ç–∞–º–∏ —Å–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –µ—Å–ª–∏ –æ–Ω–∏ —Å—Ç–æ—è—Ç –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ.

–û–ø–∏—Å–∞–Ω–∏–µ:
–ü–æ—Å–ª–µ –∫–∞–∂–¥–æ–≥–æ –ø—Ä–æ—Ö–æ–¥–∞ —Å–∞–º—ã–π –±–æ–ª—å—à–æ–π —ç–ª–µ–º–µ–Ω—Ç ¬´–≤—Å–ø–ª—ã–≤–∞–µ—Ç¬ª –≤ –∫–æ–Ω–µ—Ü –º–∞—Å—Å–∏–≤–∞.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –ü—Ä–æ—Ö–æ–¥–∏–º –ø–æ –º–∞—Å—Å–∏–≤—É –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑.
2. –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º —Å–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã.
3. –ú–µ–Ω—è–µ–º –∏—Ö –º–µ—Å—Ç–∞–º–∏, –µ—Å–ª–∏ –ø–æ—Ä—è–¥–æ–∫ –Ω–µ–≤–µ—Ä–Ω—ã–π.
4. –ü–æ—Å–ª–µ –∫–∞–∂–¥–æ–≥–æ –ø—Ä–æ—Ö–æ–¥–∞ —Å–∞–º—ã–π –±–æ–ª—å—à–æ–π —ç–ª–µ–º–µ–Ω—Ç –æ–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è –≤ –∫–æ–Ω—Ü–µ.
5. –ü–æ–≤—Ç–æ—Ä—è–µ–º, –ø–æ–∫–∞ –º–∞—Å—Å–∏–≤ –Ω–µ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: 
–•—É–¥—à–∏–π –∏ —Å—Ä–µ–¥–Ω–∏–π —Å–ª—É—á–∞–π: O(n¬≤), –ª—É—á—à–∏–π —Å–ª—É—á–∞–π: O(n).

–ü—Ä–∏–º–µ—Ä:
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

print(bubble_sort([64, 34, 25, 12, 22, 11, 90]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[11, 12, 22, 25, 34, 64, 90]`



–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—Å—Ç–∞–≤–∫–∞–º–∏ (Insertion Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:
–ê–ª–≥–æ—Ä–∏—Ç–º –≤—Å—Ç–∞–≤–∫–∞–º–∏ —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—É—é —á–∞—Å—Ç—å –º–∞—Å—Å–∏–≤–∞, –≤—Å—Ç–∞–≤–ª—è—è –∫–∞–∂–¥—ã–π –Ω–æ–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç –Ω–∞ —Å–≤–æ—ë –º–µ—Å—Ç–æ.

–û–ø–∏—Å–∞–Ω–∏–µ: 
–ö–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç—Å—è —Å –ø—Ä–µ–¥—ã–¥—É—â–∏–º–∏ –∏ –≤—Å—Ç–∞–≤–ª—è–µ—Ç—Å—è –≤ –ø—Ä–∞–≤–∏–ª—å–Ω—É—é –ø–æ–∑–∏—Ü–∏—é.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –ù–∞—á–∏–Ω–∞–µ–º —Å–æ –≤—Ç–æ—Ä–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞.
2. –°–æ—Ö—Ä–∞–Ω—è–µ–º –µ–≥–æ –≤–æ –≤—Ä–µ–º–µ–Ω–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π.
3. –°–¥–≤–∏–≥–∞–µ–º –≤—Å–µ –±–æ–ª—å—à–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤–ø—Ä–∞–≤–æ.
4. –í—Å—Ç–∞–≤–ª—è–µ–º —ç–ª–µ–º–µ–Ω—Ç –≤ –Ω–∞–π–¥–µ–Ω–Ω—É—é –ø–æ–∑–∏—Ü–∏—é.
5. –ü–æ–≤—Ç–æ—Ä—è–µ–º –¥–ª—è –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤.

–°–ª–æ–∂–Ω–æ—Å—Ç—å:
–•—É–¥—à–∏–π –∏ —Å—Ä–µ–¥–Ω–∏–π —Å–ª—É—á–∞–π: O(n¬≤), –ª—É—á—à–∏–π —Å–ª—É—á–∞–π: O(n).

–ü—Ä–∏–º–µ—Ä:
```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

print(insertion_sort([15, 8, 42, 4, 23, 16]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[4, 8, 15, 16, 23, 42]`



–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ —Å–ª–∏—è–Ω–∏–µ–º (Merge Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:
–ê–ª–≥–æ—Ä–∏—Ç–º ¬´—Ä–∞–∑–¥–µ–ª—è–π –∏ –≤–ª–∞—Å—Ç–≤—É–π¬ª, –∫–æ—Ç–æ—Ä—ã–π –¥–µ–ª–∏—Ç –º–∞—Å—Å–∏–≤ –Ω–∞ —á–∞—Å—Ç–∏, —Å–æ—Ä—Ç–∏—Ä—É–µ—Ç –∏—Ö –∏ –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –≤ –æ–¥–∏–Ω –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π.

–û–ø–∏—Å–∞–Ω–∏–µ:
–ú–∞—Å—Å–∏–≤ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ –¥–µ–ª–∏—Ç—Å—è –ø–æ–ø–æ–ª–∞–º, –∫–∞–∂–¥–∞—è —á–∞—Å—Ç—å —Å–æ—Ä—Ç–∏—Ä—É–µ—Ç—Å—è –∏ —Å–ª–∏–≤–∞–µ—Ç—Å—è.

**–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:**
1. –†–∞–∑–¥–µ–ª—è–µ–º –º–∞—Å—Å–∏–≤ –ø–æ–ø–æ–ª–∞–º.
2. –†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä—É–µ–º –ª–µ–≤—É—é –∏ –ø—Ä–∞–≤—É—é –ø–æ–ª–æ–≤–∏–Ω—ã.
3. –û–±—ä–µ–¥–∏–Ω—è–µ–º –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ —á–∞—Å—Ç–∏ –≤ –æ–¥–∏–Ω –º–∞—Å—Å–∏–≤.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: 
O(n log n)

–ü—Ä–∏–º–µ—Ä:
```python
def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr)//2
        left = arr[:mid]
        right = arr[mid:]

        merge_sort(left)
        merge_sort(right)

        i = j = k = 0
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1

        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1

        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1
    return arr

print(merge_sort([38, 27, 43, 3, 9, 82, 10]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[3, 9, 10, 27, 38, 43, 82]`



–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –®–µ–ª–ª–∞ (Shell Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:
–£—Å–æ–≤–µ—Ä—à–µ–Ω—Å—Ç–≤–æ–≤–∞–Ω–Ω–∞—è –≤–µ—Ä—Å–∏—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏ –≤—Å—Ç–∞–≤–∫–∞–º–∏, –∏—Å–ø–æ–ª—å–∑—É—é—â–∞—è —à–∞–≥ (gap) –º–µ–∂–¥—É —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º—ã–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏.

–û–ø–∏—Å–∞–Ω–∏–µ:  
–≠–ª–µ–º–µ–Ω—Ç—ã —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –∏ –ø–µ—Ä–µ—Å—Ç–∞–≤–ª—è—é—Ç—Å—è –Ω–∞ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω–æ–º —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–∏, —à–∞–≥ –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ —É–º–µ–Ω—å—à–∞–µ—Ç—Å—è.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –í—ã–±–∏—Ä–∞–µ–º –Ω–∞—á–∞–ª—å–Ω—ã–π —à–∞–≥ gap = n/2.
2. –°–æ—Ä—Ç–∏—Ä—É–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã —Å —ç—Ç–∏–º —à–∞–≥–æ–º –º–µ—Ç–æ–¥–æ–º –≤—Å—Ç–∞–≤–æ–∫.
3. –£–º–µ–Ω—å—à–∞–µ–º gap –≤–¥–≤–æ–µ.
4. –ü–æ–≤—Ç–æ—Ä—è–µ–º –¥–æ gap = 1.

–°–ª–æ–∂–Ω–æ—Å—Ç—å:
–°—Ä–µ–¥–Ω—è—è ‚Äî O(n log n), —Ö—É–¥—à–∞—è ‚Äî O(n¬≤).

–ü—Ä–∏–º–µ—Ä:
```python
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

print(shell_sort([23, 12, 1, 8, 34, 54, 2, 3]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[1, 2, 3, 8, 12, 23, 34, 54]`



 –ë—ã—Å—Ç—Ä–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ (Quick Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:
–ê–ª–≥–æ—Ä–∏—Ç–º "—Ä–∞–∑–¥–µ–ª—è–π –∏ –≤–ª–∞—Å—Ç–≤—É–π", –∫–æ—Ç–æ—Ä—ã–π –≤—ã–±–∏—Ä–∞–µ—Ç –æ–ø–æ—Ä–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏ –¥–µ–ª–∏—Ç –º–∞—Å—Å–∏–≤ –Ω–∞ –¥–≤–µ —á–∞—Å—Ç–∏: –º–µ–Ω—å—à–∏–µ –∏ –±–æ–ª—å—à–∏–µ –æ–ø–æ—Ä–Ω–æ–≥–æ.

–û–ø–∏—Å–∞–Ω–∏–µ:  
–ú–∞—Å—Å–∏–≤ –¥–µ–ª–∏—Ç—Å—è –ø–æ pivot –∏ —Å–æ—Ä—Ç–∏—Ä—É–µ—Ç—Å—è —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –í—ã–±–∏—Ä–∞–µ–º –æ–ø–æ—Ä–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç.
2. –†–∞–∑–¥–µ–ª—è–µ–º –º–∞—Å—Å–∏–≤ –Ω–∞ –º–µ–Ω—å—à–∏–µ –∏ –±–æ–ª—å—à–∏–µ.
3. –†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä—É–µ–º –æ–±–µ —á–∞—Å—Ç–∏.
4. –û–±—ä–µ–¥–∏–Ω—è–µ–º –∏—Ö.

–°–ª–æ–∂–Ω–æ—Å—Ç—å:
–°—Ä–µ–¥–Ω–∏–π —Å–ª—É—á–∞–π: O(n log n), —Ö—É–¥—à–∏–π: O(n¬≤).

–ü—Ä–∏–º–µ—Ä:
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr)//2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

print(quick_sort([24, 15, 38, 2, 19, 41, 8]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[2, 8, 15, 19, 24, 38, 41]`



üèóÔ∏è –ü–∏—Ä–∞–º–∏–¥–∞–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ (Heap Sort)
–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:
–ü–∏—Ä–∞–º–∏–¥–∞–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å—Ç—Ä—É–∫—Ç—É—Ä—É –¥–∞–Ω–Ω—ã—Ö ¬´–∫—É—á–∞¬ª (heap) –¥–ª—è —É–ø–æ—Ä—è–¥–æ—á–∏–≤–∞–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤.

–û–ø–∏—Å–∞–Ω–∏–µ:
–ú–∞—Å—Å–∏–≤ –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç—Å—è –≤ max-–∫—É—á—É, –∑–∞—Ç–µ–º –∏–∑–≤–ª–µ–∫–∞—é—Ç—Å—è –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –ø–æ –æ–¥–Ω–æ–º—É.

–ü–æ—à–∞–≥–æ–≤–∞—è —Ä–∞–±–æ—Ç–∞:
1. –ü–æ—Å—Ç—Ä–æ–∏—Ç—å max-heap –∏–∑ –º–∞—Å—Å–∏–≤–∞.
2. –ü–æ–º–µ—Å—Ç–∏—Ç—å –Ω–∞–∏–±–æ–ª—å—à–∏–π —ç–ª–µ–º–µ–Ω—Ç –≤ –∫–æ–Ω–µ—Ü.
3. –£–º–µ–Ω—å—à–∏—Ç—å —Ä–∞–∑–º–µ—Ä –∫—É—á–∏ –∏ –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏—Ç—å —Å–≤–æ–π—Å—Ç–≤–æ max-heap.
4. –ü–æ–≤—Ç–æ—Ä–∏—Ç—å, –ø–æ–∫–∞ –∫—É—á–∞ –Ω–µ –ø—É—Å—Ç–∞.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: 
O(n log n)

–ü—Ä–∏–º–µ—Ä:
```python
def heapify(arr, n, i):
    largest = i
    l = 2 * i + 1
    r = 2 * i + 2

    if l < n and arr[l] > arr[largest]:
        largest = l
    if r < n and arr[r] > arr[largest]:
        largest = r
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]
        heapify(arr, i, 0)
    return arr

print(heap_sort([4, 10, 3, 5, 1]))
```
–†–µ–∑—É–ª—å—Ç–∞—Ç: `[1, 3, 4, 5, 10]`
